Class: Card

Card(id, name, imageURL, type, hp, evolution)

new Card(id, name, imageURL, type, hp, evolution)

Each card has the required field of an ID and optional fields of name, imageURL, type, hp, and evolution.

Parameters:
Name Type Description
id int

The unique ID for a card.
name string

The pokemon's name.
imageURL string

The URL from indexDB of the card image.
type string

The pokemon's specific type.
hp int

The amount of health points the pokemon has.'
evolution string

The evolution stage of a pokemon (Basic, Stage 1, Stage 2).
